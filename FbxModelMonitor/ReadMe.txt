================================================================================
    MFC (MICROSOFT FOUNDATION CLASS) ライブラリ : FbxModelMonitor プロジェクトの概要
===============================================================================

この FbxModelMonitor アプリケーションは、アプリケーション ウィザードによって作成されました。このアプリケーションは、Microsoft Foundation Classes の使用の基本を示しているだけでなく、アプリケーションの記述の出発点でもあります。

このファイルには、FbxModelMonitor アプリケーションを構成する各ファイルの内容の概略が記述されています。

FbxModelMonitor.vcproj
    これは、アプリケーション ウィザードで生成された VC++ プロジェクトのメイン プロジェクト ファイルです。
    ファイルを生成した Visual C++ のバージョンに関する情報と、アプリケーション ウィザードで選択されたプラットフォーム、構成、およびプロジェクト機能に関する情報が含まれています。

FbxModelMonitor.h
    これは、アプリケーションのメイン ヘッダー ファイルです。他のプロジェクト固有のヘッダー (Resource.h など) をインクルードしており、CFbxModelMonitorApp アプリケーション クラスを宣言しています。

FbxModelMonitor.cpp
    これは、メインのアプリケーション ソース ファイルで、アプリケーション クラス CFbxModelMonitorApp が含まれています。

FbxModelMonitor.rc
    これは、プログラムが使用するすべての Microsoft Windows リソースの一覧です。RES サブディレクトリに格納されるアイコン、ビットマップ、およびカーソルをインクルードしています。このファイルは、Microsoft Visual C++ で直接編集できます。プロジェクト リソースは 1041 にあります。

res\FbxModelMonitor.ico
    これは、アプリケーションのアイコンとして使用されるアイコン ファイルです。このアイコンは、メイン リソース ファイル FbxModelMonitor.rc にインクルードされます。

res\FbxModelMonitor.rc2
    このファイルには、Microsoft Visual C++ で編集しないリソースが含まれています。リソース エディタで編集できないすべてのリソースは、このファイルに含める必要があります。

/////////////////////////////////////////////////////////////////////////////

メイン フレーム ウィンドウ用 :
    プロジェクトには、標準の MFC インターフェイスが含まれています。

MainFrm.h, MainFrm.cpp
    これらのファイルには、フレーム クラス CMainFrame が含まれています。このフレーム クラスは、
    CFrameWnd から派生したもので、すべての SDI フレーム機能を制御します。

res\Toolbar.bmp
    このビットマップ ファイルは、ツール バーに並べて表示されるイメージを作成するために使用されます。
    最初のツール バーおよびステータス バーは、CMainFrame クラスで構築されます。このツール バー ビットマップをリソース エディタで編集し、FbxModelMonitor.rc 内の IDR_MAINFRAME TOOLBAR 配列を更新して、ツール バー ボタンを追加します。
/////////////////////////////////////////////////////////////////////////////

アプリケーション ウィザードによって、次の 1 つのドキュメント タイプと 1 つの表示が生成されます。

FbxModelMonitorDoc.h, FbxModelMonitorDoc.cpp - ドキュメント
    これらのファイルには、CFbxModelMonitorDoc クラスが含まれています。これらのファイルを編集して、独自のドキュメント データを追加したり、ファイルの保存と読み込みを行ったりできます (CFbxModelMonitorDoc::Serialize を使用)。

FbxModelMonitorView.h, FbxModelMonitorView.cpp - ドキュメントの表示
    これらのファイルには、CFbxModelMonitorView クラスが含まれています。
    CFbxModelMonitorView オブジェクトは、CFbxModelMonitorDoc オブジェクトの表示に使用します。





/////////////////////////////////////////////////////////////////////////////

その他の機能 :

ActiveX コントロール
    アプリケーションには、ActiveX コントロールを使用するためのサポートが含まれています。

印刷および印刷プレビューのサポート
    MFC ライブラリから CView クラスにメンバ関数を呼び出すことにより、アプリケーション ウィザードで生成されたコードで、印刷、印刷の設定、および印刷プレビューの各コマンドを扱うことができます。

分割ウィンドウ
    アプリケーション ウィザードで、アプリケーション ドキュメントの分割ウィンドウがサポートされるようになりました。

その他の標準ファイル :

StdAfx.h, StdAfx.cpp
    これらのファイルは、プリコンパイル済みヘッダー (PCH) ファイル FbxModelMonitor.pch とプリコンパイル済み型ファイル StdAfx.obj をビルドするために使用されます。

Resource.h
    これは、新しいリソース ID を定義する標準のヘッダー ファイルです。
    このファイルの読み込みおよび更新は、Microsoft Visual C++ で行います。

FbxModelMonitor.manifest
	アプリケーション マニフェスト ファイルは Windows XP で、	side-by-side アセンブリの特定のバージョンに関するアプリケーションの依存関係を説明するために使用されます。ローダーは、この	情報を使用して、アセンブリ キャッシュから適切なアセンブリを、またはアプリケーションからプライベート アセンブリを	読み込みます。アプリケーション マニフェストは再頒布用に、	アプリケーションの実行可能ファイルと同じフォルダにインストールされる外部 .manifest ファイルとして含まれているか、	またはリソースの形式で実行可能ファイル内に含まれています。
